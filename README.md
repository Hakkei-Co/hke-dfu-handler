# hke-dfu-handler

## Usage
```
$ pnpm install
$ pnpm run dev
```

## Installation

```
$ npm install --save @hakkei/dfu-handler
```

## Markup

```html
<div class="p-5" id="app">
  <dfu-handler></dfu-handler>
</div>
```

With script tag
```html
<script type="module" src="./node_modules/@hakkei/dfu-handler/dist/assets/index.js">
```
